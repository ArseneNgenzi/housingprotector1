# housingprotector1

### Steps

1. clone the project

```
$ git clone https://github.com/ArseneNgenzi/housingprotector1.git
```

2. Go to root folder:

```
$ cd housingprotector1
```

3. Install node modules:

```
$ npm install
```

6. run the application:

```
$ node app
```

## For better developer experience, run the server with nodemon by typing:

```
$ npm run dev
```
